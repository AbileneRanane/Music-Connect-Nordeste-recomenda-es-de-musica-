# Music-Connect-Nordeste-recomenda-es-de-musica-
Music Connect Nordeste é um projeto desenvolvido como desafio de criação de um sistema de recomendação de músicas utilizando grafos com o banco de dados Neo4j. O objetivo é mostrar como informações musicais — usuários, artistas, gêneros e musicas,  no qual podem ser representadas em um grafo e exploradas para gerar recomendações inteligentes.
